# Privacy Policy

**@rexeus/agentic** does not collect, store, or transmit any user data.

The plugin runs entirely within your local Claude Code environment. All agents,
commands, skills, and hooks operate on local files and produce local output.
No data is sent to external servers, APIs, or third-party services.

The only external interaction occurs when you explicitly invoke commands that
use installed CLI tools (such as `gh` for Pull Requests or `git` for version
control). These tools connect to services you have already authenticated with
and are not controlled by this plugin.
